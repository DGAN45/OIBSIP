# 📊 Unemployment Analysis in India During the Pre & Post COVID Period

## 📌 Project Overview

This project performs **Exploratory Data Analysis (EDA)** on India's unemployment dataset to understand how unemployment rates changed before and after the COVID-19 pandemic. The analysis identifies regional trends, monthly variations, and the overall impact of lockdown on employment across different Indian states.

The project uses Python libraries for data cleaning, visualization, and statistical analysis.

---

## 🎯 Objectives

- Analyze unemployment trends across Indian states.
- Compare unemployment rates before and after COVID-19.
- Identify the states with the highest and lowest unemployment rates.
- Perform month-wise unemployment trend analysis.
- Visualize unemployment using charts and heatmaps.
- Draw meaningful insights from the dataset.

---

## 📂 Dataset

**Dataset Used:** Unemployment in India Dataset

The dataset contains information such as:

- Region (State)
- Date
- Frequency
- Estimated Unemployment Rate (%)
- Estimated Employed
- Estimated Labour Participation Rate (%)
- Area (Urban/Rural)
